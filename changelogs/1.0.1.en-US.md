# Meow Anti-Xray 1.0.1 Changelog

This is a maintenance release focused on clearer runtime diagnostics for server owners.

## Improvements

- `/antixray status` now shows the active loader, mod version, and Minecraft version.
- `/antixray profile` now reports async rewrite enablement, worker count, queue size, total capacity, in-flight count, and available permits.
- `/antixray profile` now tracks sync chunk sends and sync send rate, making async backpressure fallback easier to spot.
- Local `.playwright-cli/` test cache is now ignored so it does not pollute the working tree.

## Verification

- Fabric and NeoForge unit tests passed.
- Fabric and NeoForge dual-loader build passed.
