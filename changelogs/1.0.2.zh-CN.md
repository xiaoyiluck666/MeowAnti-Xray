# Meow Anti-Xray 1.0.2 更新日志

这是一个稳定修复版本，重点修复 Fabric 端区块包重写在真实 local palette 场景下的崩溃，并进一步对齐 Paper Anti-Xray 的 engine mode 1 行为。

## 修复

- 修复 Fabric 端玩家加载区块时可能出现的 `MissingPaletteEntryException` 崩溃。
- 修正 local palette 读取逻辑：实时读取 `LevelChunkSection` 时现在使用真实 palette 大小，而不是 4096 个区块内方块索引数量。
- 修正 engine mode 1 / HIDE 的自然替换规则：主世界按高度选择 `stone` / `deepslate`，下界使用 `netherrack`，末地使用 `end_stone`，更贴近 Paper 行为。

## 验证

- 新增 local palette 回归测试，覆盖 Fabric 崩溃路径。
- 新增 engine mode 1 自然替换 golden case 测试。
- Fabric 8 个假网络玩家 RCON spectator 跑图 91 秒通过：`15833` chunks，约 `173.94 chunks/s`，客户端错误 `0`。
- Fabric 与 NeoForge 单元测试通过。
- Fabric 与 NeoForge 双 loader 构建通过。
