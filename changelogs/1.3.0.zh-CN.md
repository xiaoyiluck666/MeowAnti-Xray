# Meow Anti-Xray 1.3.0 更新日志

这是面向 Minecraft 26.2 的兼容与验证版本，重点是升级双 loader 依赖、校准官方版本信息，并补齐 26.2 下的真实压测结果。

## 兼容性

- 支持 Minecraft `26.2`，Java 语言级别保持 `25`。
- Fabric 侧升级到 Fabric Loader `0.19.3` 与 Fabric API `0.153.0+26.2`。
- NeoForge 侧升级到 `26.2.0.7-beta`。
- Fabric metadata 现在限制为 `>=26.2 <26.3`；NeoForge Minecraft 范围为 `[26.2,26.3)`。

## 压测工具

- `tools/mc-2612-load-runner.js` 新增 `--mc-version 26.2` 协议 profile，默认使用 Minecraft 26.2 / protocol `776`。
- 保留 `26.1.2` / protocol `775` profile，便于回归旧版本。
- 修正 26.2 configuration 阶段 custom payload 处理：不再把服务端插件消息原样回发，避免服务端按新的 serverbound payload codec 解码失败。

## 压测结果

本轮使用默认 `async-queue-size=16`，8 个网络假玩家通过 RCON spectator 传送持续制造区块发送压力：

| Loader | 时长 | 区块包 | 吞吐 | 客户端错误 | rewrite 平均耗时 | sync fallback |
| --- | ---: | ---: | ---: | ---: | ---: | ---: |
| Fabric | `92s` | `32,893` | `357.49 chunks/s` | `0` | `2.839ms` | `42.0%` |
| NeoForge | `92s` | `35,593` | `386.86 chunks/s` | `0` | `2.719ms` | `39.0%` |

`pressure=saturated` 在这种高压跑图下是预期信号，表示默认队列背压正在限制异步任务堆积，优先保护内存稳定。

## Paper 对齐

- 已复查 PaperMC/Paper `main`，远端 HEAD 仍为 `76d2ac758cb3abe75aceefa88207443768f585c6`，与项目既有 Paper anti-xray parity 基线一致。
- Paper anti-xray 默认配置、hidden/replacement 逻辑与权限节点未发现新的上游变化，本轮不需要额外调整核心反矿透行为。
